Example

This page demonstrates how to annotate step by step.

1) Check the target object shown in mutiple video frames.
2) Watch the video and provide a description about what the target object is doing in the video. You have to annotate it in two ways:
* Describe freely in ONE sentence, explicitly specifying the category of the object (like car, dog, etc.) and the action of what the object is doing (like jumping, walking, etc.).
* Describe by selecting WORDS, indicating the COLOR, OBJECT, ACTION and POSITION of the object. COLOR and POSITION can be empty if there is no suitable options.
